Detailed Description

Solution of Vehicle Routing Problem.

Scene contains schedules for the resources, i.e. a set of order assignments to the resources. Orders and resources themselves are defined by the scene context. Single scene context can have multiple associated scenes (e.g. different solutions for the same set of input data).

const SceneQueries & Scheduler::Scene::query ( ) const

Returns the interface to execure scene queries.

Scene query is an interface to retrieve some kind of data from the scene (e.g. find work stop associated with the operation or calculate total vehicle demand) Queries can use caching to boost execution performance so it's the preferred way to retrieve supported information
