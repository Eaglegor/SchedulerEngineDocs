template<typename Alg , typename... Args>
Alg& Scheduler::AlgorithmsManager::createAlgorithm ( Args &&...  args)
inline

Creates new algorithm.

Algorithm is created using the underlying factory and the reference to the newly created instance is returned. The lifetime of created algorithms is managed by AlgorithmsManager. Algorithms that weren't explicitly destroyed by the call to the destroyAlgorithm method will be automatically destroyed when the AlgorithmsManager is destroyed.
