Detailed Description

template<typename CollectionType, typename AdoptedType>
class Scheduler::RangeTypeAdapter< CollectionType, AdoptedType >

This class wraps the container object and casts the elements of underlying container to the specified type on retrieval.

Template Parameters
CollectionTypeType of the container
AdoptedTypeType to cast the elements to
Note
AdoptedType must be CollectionType::value_type or the class derived from CollectionType::value_type
