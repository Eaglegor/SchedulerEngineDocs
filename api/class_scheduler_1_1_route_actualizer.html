Detailed Description

Updates next route of the stop.

Internally this class uses dirty flag to reduce recalculations. This flag must be set explicitly to notify actualizer that the object has changed and recalculation must be performed upon next call to actualize()

void Scheduler::RouteActualizer::actualize ( ) const

Recalculates next route for the stop.

If dirty flag is set to false this method just returns. If dirty flag is set to true recalculation is performed:

  1. RouteActualizationAlgorithm is retrieved from the ScheduleActualizationModel
  2. Algorithm is called to update the next route of the stop
  3. Dirty flag is set to false
