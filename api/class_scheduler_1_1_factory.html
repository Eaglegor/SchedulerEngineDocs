Detailed Description

template<typename BaseClass>
class Scheduler::Factory< BaseClass >

Templated factory class used to create the objects inside the scheduler engine.

Internally this factory uses pool allocator to allocate memory so the size of create objects is limited by a fixed number passed on factory construction

Template Parameters
BaseClassThe type of base class for the objects created by this factory. Base class and the classes derived from base class can be instantiated using this factory
