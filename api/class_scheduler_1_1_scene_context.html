Detailed Description

Vehicle Routing Problem task definition.

Scene context is a shared immutable part of VRP solution. It contains orders, resources (performers and vehicles), locations, attributes, depots. Scene context doesn't contain any assignments of orders to the resources so it doesn't represent the solution.
