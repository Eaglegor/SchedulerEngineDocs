Detailed Description

Order that has to be completed.

Order is a sequence of operations that have to be performed to satisfy the customer. Orders can have different types: deliveries, collections, services.

Three type of operations are associated with order:

  1. start operation (optional) - base operation that has to be perfomed before starting the run
  2. work operation (mandatory) - field operation that has to be performed at the customer's location
  3. end operation (optional) - base operation that has to be perfomed after finishing the run
